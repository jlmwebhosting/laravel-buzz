Laravel Buzz Bundle

Buzz is a lightweight PHP 5.3 library for issuing HTTP requests. This bundle simply wraps the Buzz library.

Installation

You can install this bundle by running the following CLI command:

$php artisan bundle:install buzz

Bundle Registration

Add the following to your application/bundles.php file:

'buzz' => array(
    'auto' => true,
),
